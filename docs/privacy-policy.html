Privacy Policy

LetsCloud Infrastructure Manager GPT

Last Updated: December 2024
Effective Date: December 2024

1. Introduction

This Privacy Policy describes how the LetsCloud Infrastructure Manager GPT ("we", "our", or "us") handles information when you use our GPT service for managing cloud infrastructure on the LetsCloud platform.

This GPT is designed to help users manage their cloud infrastructure through natural language commands, including server management, SSH key management, snapshot operations, and account information retrieval.

2. Data Collection and Usage

2.1 What We Do NOT Collect

  • Personal Data: We do not collect, store, or process any personal information
  • API Keys: Your LetsCloud API keys are never stored by this GPT
  • Server Data: Your server information is not stored or cached
  • Conversation History: We do not maintain conversation logs

2.2 What We Do Collect

  • Usage Analytics: Basic usage statistics through OpenAI's platform (subject to OpenAI's privacy policy)
  • Error Logs: Technical error information for service improvement (no personal data)

3. API Key Handling

Important: API keys are sensitive information. Please keep your LetsCloud API key secure and never share it with others.

3.1 How API Keys Are Used

  • Authentication: Your API key is used only for authenticating requests to the LetsCloud API
  • No Storage: API keys are never stored, cached, or logged
  • Secure Transmission: All API requests are transmitted securely via HTTPS
  • Temporary Use: API keys are used only during active conversations

3.2 API Key Security

  • API keys are transmitted directly to LetsCloud's secure API endpoints
  • No intermediate storage or processing of API keys
  • Each user must provide their own API key for authentication
  • API keys are not shared between users

4. Third-Party Services

4.1 OpenAI GPT Platform

This GPT operates on OpenAI's GPT platform. Your interactions are subject to OpenAI's Privacy Policy and Terms of Service.

4.2 LetsCloud API

This GPT integrates with the LetsCloud API (https://core.letscloud.io/api) for infrastructure management. API requests are subject to LetsCloud's Privacy Policy and Terms of Service.

4.3 Data Flow

  • User → OpenAI GPT Platform → LetsCloud API
  • LetsCloud API → OpenAI GPT Platform → User
  • No data is stored in our systems

5. Data Security

5.1 Security Measures

  • HTTPS Encryption: All data transmission is encrypted
  • No Local Storage: We do not maintain any local data storage
  • Secure APIs: All API communications use industry-standard security
  • Access Control: Each user controls their own API access

5.2 User Responsibilities

  • Keep your LetsCloud API key secure and confidential
  • Do not share your API key with others
  • Monitor your API usage and account activity
  • Report any security concerns immediately

6. Your Rights and Choices

6.1 Data Control

  • You control all your data through your LetsCloud account
  • You can revoke API access at any time
  • You can delete your LetsCloud account and data
  • You can stop using this GPT at any time

6.2 Opt-Out Options

  • Stop using the GPT to discontinue data processing
  • Revoke your API key in your LetsCloud dashboard
  • Contact us for any privacy concerns

7. Children's Privacy

This GPT is not intended for use by children under 13 years of age. We do not knowingly collect personal information from children under 13.

8. International Data Transfers

This GPT operates globally and may process data in different countries. All data processing follows applicable privacy laws and regulations.

9. Changes to This Privacy Policy

We may update this Privacy Policy from time to time. We will notify users of any material changes by updating the "Last Updated" date at the top of this policy.
