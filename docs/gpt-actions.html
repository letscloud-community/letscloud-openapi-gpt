What are GPT Actions?

GPT Actions allow you to connect ChatGPT and other AI assistants directly to your LetsCloud infrastructure. Instead of writing complex scripts or using command-line tools, you can simply talk to your AI assistant to manage your cloud resources.

üöÄ Quick Setup Guide

Step 1: Get Your API Key

First, you'll need your LetsCloud API key:

  1. Log into your LetsCloud account
  2. Navigate to API settings
  3. Generate a new API key or copy your existing one

Step 2: Configure GPT Actions

In your ChatGPT or OpenAI platform:

  1. Go to GPT Actions configuration
  2. Click "Add Action"
  3. Import the OpenAPI specification from this repository
  4. Configure authentication with your API key

Step 3: Start Using

Now you can use natural language commands like:

  • "Create a new web server with Ubuntu 22.04"
  • "List all my servers and their status"
  • "Create a backup snapshot of my production server"
  • "Optimize my infrastructure costs"
